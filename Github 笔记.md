
git init (创建版本库，初始化本地仓库)

git add filename   (把新建的文件添加到版本库)

git commit -m "comment"   (提交到github上，必须要填写注释)

git status (查看文件修改状态,工作区的状态[-s 用精简状态显示。M位置不同代边不同的状态])

git stash (保存当前的工作进度)

git add . (添加本地文件更改) -A（添加删除更改）

git push
